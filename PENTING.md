# PENTING — Baca Ini Duluan

Project ini dikerjakan bergantian dari beberapa akun/sesi AI yang berbeda (bukan
cuma satu). Supaya nggak bingung dan nggak ngulang kesalahan yang sudah pernah
kejadian, **sebelum mulai kerja apapun**, baca 3 file ini secara berurutan:

1. **`README.md`** — instruksi wajib buat AI yang kerja di project ini (alur
   kerja, aturan zip per bagian, dll)
2. **`CHANGELOG.md`** — riwayat semua pembaruan dari awal sampai versi
   terakhir. Baca dari paling atas (versi terbaru) biar tahu progress terkini.
3. **`RENCANA.md`** — rencana kerja yang **belum** dieksekusi + checklist-nya.
   **Ini yang paling penting** karena isinya arahan langkah selanjutnya yang
   harus diikuti persis, termasuk aturan ketat "1 section per langkah, jangan
   digabung".

Kalau ada perubahan yang butuh penjelasan lebih detail (alasan teknis, proses
coba-coba, keputusan kenapa begini bukan begitu), itu ada di folder
**`catatan/`** — dibagi per topik, urut angka (01, 02, 03, dst). Cek folder ini
kalau CHANGELOG.md merujuk ke catatan tertentu atau kalau butuh konteks lebih
dalam soal satu perubahan.

## Ringkas alurnya

```
PENTING.md (kamu di sini)
   │
   ├─ README.md      → aturan main
   ├─ CHANGELOG.md    → sudah sampai mana
   ├─ RENCANA.md      → mau ngapain selanjutnya
   └─ catatan/        → detail teknis kalau perlu
```

Nggak perlu baca semua isi project sekaligus dari awal — cukup 3 file di atas
(README, CHANGELOG, RENCANA), baru catatan/ kalau dirujuk atau dibutuhkan.
Setelah paham, ikuti instruksi di `RENCANA.md` persis seperti yang tertulis,
termasuk urutan kerja dan aturan konfirmasi user sebelum lanjut ke langkah
berikutnya.
